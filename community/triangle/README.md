# Triangle

Write a program that takes three integer values from the user input and outputs true if a triangle can be constructed with the sides of given length.

For example:
```
Input: 3, 4, 5
Output: True

Input: 10, 2, 9
Output: True

Input: 7, 9, 1
Output: False
```
Bonus: Visualize the triangles (web or terminal).
